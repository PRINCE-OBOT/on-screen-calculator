# On-Screen Calculator

An On-Screen Calculator project built as part of [The Odin Project Calculator Project](https://www.theodinproject.com/lessons/node-path-javascript-calculator) curriculum to practice JavaScript logic, DOM manipulation, and handling user input through an interactive calculator UI.

---

## Features

* Basic arithmetic operations (+, −, ×, ÷)
* Clear and reset functionality
* Decimal number support
* Keyboard or button-based input (if implemented)
* Real-time display updates
* Error handling for invalid operations
* Reset calculation

---

## Tech Stack

* JavaScript
* HTML
* CSS

---

## What I Learned

* Building core calculator logic from scratch
* Handling chained operations
* Managing input state
* Preventing invalid calculations (like divide by zero)

---

## Challenges

* Handling multiple sequential operations correctly
* Managing floating point precision issues
* Preventing invalid input states
* Toggling a number negative or positive
